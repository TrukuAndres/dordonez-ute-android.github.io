¿Qué son los menús?

Son un componente que encontramos comunmente en la intefaz del usuario.

Un menú presenta al usuario opciones dentro de las actividades.



Menú de opciones y barra de app

El menú de opciones es donde se encuentran los elementos principales del menú como, "Buscar", "Configuraciones","Ayuda y comentarios", etc.





Menú contextual y modo de acción contextual

Este es el menú que aparece cuando el usuario hace un click largo en un elemento.


Proporciona acciones que afectarán al contenido seleccionado. Como puede ser un texto que queremos copiar o una imagen que queremos descargar.

Como un ejemplo podemos observar el copiar un texto de la aplicación de Google.





Menú emergente

Muestra una lista de elementos en una lista vertical que está anclada a la vista que invocó el menú.


Sirve para proporcionar una ampliación de acciones relacionadas con contenido específico.

Las acciones en un menú emergente no deben afectar directamente al contenido correspondiente, de esto se encargan los menús contextuales





Definir un menú en XML

Ya que Android nos proporciona un formato XML estándar que permite definir los elementos de un menú.


En lugar de incorporar un menú en el código de la actividad, debes definir un menú y todos los elementos en un recurso de menú XML.

< menu> Define un Menú, que es un contenedor para elementos como item y group.

< item> Crea un MenuItem, que representa un único elemento en un menú.

< group> Contenedor opcional e invisible para elementos item. Permite categorizar los elementos de menú para que compartan propiedades, como el estado de una actividad o visibilidad.

Ejemplo de menú en XML:




"android:icon" El ID de un recurso que es exclusivo del elemento

"android:icon" Referencia a un elemento de diseño para usar como el ícono del elemento.

"android:title" Referencia a una string para usar como el título del elemento.

"android:showAsAction" Especifica cuándo y cómo el elemento debe aparecer como un elemento de acción en la barra de app.

Manejar eventos de clic

Cuando el usuario selecciona un elemento del menú de opciones el sistema llama al método onOptionsItemSelected() de la actividad. Este elemento pasa al MenuItem seleccionado.

Utiliznado getItemId() podemos hacer coincidir este ID con elementos de menú conocidos para realizar la acción.





Crear menús contextuales

Como indicamos antes el menú contextual brinda acciones que afectan a un elemento del marco contextual específico.

Estos menús pueden ser:

  • Menú contextual flotante.
  • Modo de acción contextual.

Ejemplo de los dos tipos de menú contextual. A la derecha menú contextual flotante y a la izquierda modo de acción contextual.





Menú contextual flotante

En este caso un menú aparece como una lista flotante de elementos del menú, es muy similar a un cuadro de dialogo.

A diferencia del "modo de acción contextual" aqui solo se admite un valor contextual.

Pasos para crear un menú contextual flotante

1 Registra la View con la que se debe asociar el menú contextual llamando a registerForContextMenu() y pásalo a la View.

2 Implementar el método onCreateContextMenu() en Activity.





3 Implementa onContextItemSelected().




Modo de acción contextual

En este caso no aparece una lista flotante sino que se selecciona a un contexto y en la parte superior aparecera una barra de acciones contextuales.

En este modo el usuario puede seleccionar varios elementos, como ejemplo seleccionar varios mensajes en la aplicación Whatsaap para eliminarlos.

Podemos crear modos de acción individuales o grupales.

Pasos para crear modo de acción contextual para vistas individuales.

1 Implementa la interfaz ActionMode.Callback. En los métodos de devolución de llamada se puede especificar las acciones de la barra de acciones contextuales, responder a eventos de clic en elementos de acción o manejar otros eventos del ciclo de vida para el modo de acción.





2 Llamar a startActionMode() cuando se desee mostrar la barra.




Pasos para habilitar acciones contextuales por lotes en una ListView o GridView.

1 Implementa la interfaz AbsListView.MultiChoiceModeListener y establécela para la vista de grupo con setMultiChoiceModeListener().

2 Llama a setChoiceMode() con el argumento CHOICE_MODE_MULTIPLE_MODAL.





Cómo crear un menú emergente

Tambien conocido como PopUp es un menú anclado a una View.


Proporcionar un menú de estilo de ampliación para las acciones que se relacionan con contenido específico .

Pasos para crear un PopUp

1 Crear una instancia de PopupMenu con su constructor, que toma el Context de la aplicación actual y la View a la que debe anclarse el menú.

2 Usa MenuInflater para inflar el recurso de menú hacia el objeto Menú devuelto por PopupMenu.getMenu(). Llama a PopupMenu.show()
