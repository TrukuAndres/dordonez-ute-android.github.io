¿Qué son los menús?

Son un componente que encontramos comunmente en la intefaz del usuario.

Un menú presenta al usuario opciones dentro de las actividades.



Menú de opciones y barra de app

El menú de opciones es donde se encuantran los elementos principales del menú como, "Buscar", "Configuraciones","Ayuda y comentarios", etc.





Menú contextual y modo de acción contextual

Este es menú que aparece cuando el usuario hace un click largo en un elemento.


Proporciona acciones que afectaran al contenido seleccionado. Como puede ser un texto que queremos copiar o una imagen que quermos descargar.

Como un ejemplo podemos observar el copiar un texto de la aplicación de Google.





Menú emergente

Muestra una lista de elementos en una lista vertical que está anclada a la vista que invocó el menú.


Sirve para proporcionar una ampliación de acciones relacionadas con contenido específico.

Las acciones en un menú emergente no deben afectar directamente al contenido correspondiente, de esto se encargan los menús contextuales





Definir un menú en XML

Ya que Android nos proporciona un formato XML estándar que permite definir los elementos de menú.


En lugar de incorporar un menú en el código de la actividad, debes definir un menú y todos los elementos en un recurso de menú XML.

< menu> Define un Menú, que es un contenedor para elementos como item y group.

< item> Crea un MenuItem, que representa un único elemento en un menú.

< group> Contenedor opcional e invisible para elementos item. Permite categorizar los elementos de menú para que compartan propiedades, como el estado de una actividad o visibilidad.

Ejemplo de menú en XML:




"android:icon" El ID de un recurso que es exclusivo del elemento

"android:icon" Referencia a un elemento de diseño para usar como el ícono del elemento.

"android:title" Referencia a una string para usar como el título del elemento.

"android:showAsAction" Especifica cuándo y cómo el elemento debe aparecer como un elemento de acción en la barra de app.
