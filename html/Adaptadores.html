Crear una lista con RecyclerView

El widget RecyclerView es una versión avanzada y flexible de ListView.

En el modelo RecyclerView, varios componentes diferentes trabajan juntos para mostrar sus datos. El contenedor general para su interfaz de usuario es un objeto RecyclerView que agrega a su diseño. RecyclerView se llena de vistas proporcionadas por un administrador de diseño que usted proporciona. Puede usar uno de nuestros administradores de diseño estándar (como LinearLayoutManager o GridLayoutManager), o implementar el suyo propio.

CONTROLES DE NÚMEROS

Los controles de números ofrecen una manera rápida de seleccionar un valor de un conjunto. En el estado predeterminado, un control de números muestra su valor actualmente seleccionado. Al tocar el control de números, se muestra un menú desplegable con todos los demás valores disponibles, de los cuales el usuario puede seleccionar uno.

Para rellenar el control de número con una lista de opciones, debemos especificar un SpinnerAdapter en nuestro código fuente de la Activity o el Fragment.

Las clases principales son las siguientes:

  • Spinner
  • SpinnerAdapter
  • AdapterView.OnItemSelectedListener
